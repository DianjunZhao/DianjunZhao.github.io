成功案例

2022-01-18

周娜律师团队自成立以来，始终恪守“攻刑以专，相靖予霖”的所训，给当事人带来明白、充分且优质的法律服务，多起案件均实现有效辩护，收获了当事人的赞誉及公检法的认可。

>> 某正部级干部配偶受贿、利用影响力受贿案，审查起诉阶段介入，经辩护，法院认定自首及犯罪未遂情节

>> 某小贷公司涉恶案，公诉机关指控两起诈骗事实（涉案金额均为数百万、量刑建议十年以上）、一起寻衅滋事事实，一审介入，最终法院认定被告人不构成诈骗罪，仅以寻衅滋事罪判处

>> 马某某贪污、窝藏案，侦查阶段介入，经辩护，在审查起诉阶段，公诉机关将诈骗罪（刑期应在有期徒刑十年以上）变更为贪污罪（刑期在有期徒刑三年以上、十年以下），并以贪污罪、窝藏罪起诉至法院，最终一审法院认定被告人不构成窝藏罪，仅以贪污罪判处

>> 吕某强奸案，一审判处有期徒刑十年六个月，二审介入，经辩护，二审法院改判有期徒刑五年

>> 王某非法吸收公众存款案，二审阶段介入，经辩护，二审法院作出撤销原判、发回重审的裁定

>> 董某生产、销售有毒、有害食品案，检察院提起抗诉，二审阶段介入，二审法院作出驳回抗诉、维持原判的终审裁定

>> 王某生产、销售有毒、有害食品案，公安机关量刑建议五至七年，经辩护最终判处有期徒刑三年，检察院抗诉后维持原判

>>某受贿，执行判决、裁定滥用职权，巨额财产来源不明案，审查起诉阶段介入，去掉执行判决、裁定滥用职权罪罪名

>> 吕某某非法吸收公众存款案，非吸金额11亿余元，经辩护最终判处有期徒刑四年六个月，并处罚金二十万元

>> 宋某诈骗案，一审阶段介入，法院判处缓刑

>> 胡某走私废物案，一审阶段介入，法院判处缓刑

>> 滕某侵犯公民个人信息案，审查起诉阶段介入，公安机关撤回移送审查起诉

>> 林某诈骗案，数额近200万，侦查阶段介入，检察院不予批准逮捕，公安变更强制措施为取保候审，取保候审期限一年期满前，公安机关解除取保候审
